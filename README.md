 # Snail Runner Game using Python
 A snail runner game made using python's pygame library. There are two types of enemies, flies and snails. Avoid them to get points. Jumping over them gives you a point. Colliding into them ends your game.
 the game.py contains source code and click the dist/game application to play directly. Best of Luck<br>
